# 🌧 oh!Rain
## Overview
oh!Rain 앱은 날씨 API를 통해 강수확률을 받아서 우산을 챙길지 말지에 대한 정보를 알려주는 앱입니다.

## Tech stack
- UIKit(Storyboard)
- URLSession
- UserNotifications
- MVC
